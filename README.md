# 🏦 Bank System (Python CLI)

This is a simple bank system project built with Python.  
It runs in the terminal and allows users to manage bank accounts.

## 🚀 Features

- Create account
- Show all accounts
- Search account
- Deposit money
- Withdraw money
- Delete account
- Show total money in bank

## 🧠 Concepts Used

- Functions
- Lists
- Loops (while, for)
- Conditions (if/else)
- Input validation
- Basic program structure

## ▶️ How to Run

1. Make sure Python is installed
2. Run the file:

```bash
python bank_system1.py
